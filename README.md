# game_save_viewer

A Flutter/Python project.

## Setup Project

 1. Clone the project
 2. Copy the files in assets folder into the game backup directory (eg:-D:/Games/)
 3. Run flutter run command and select windows.
 4. Edit the game backup directory in all 3 files when changing directories.(python file,config file,dart file)

## Python Script

``` python backup_tool.py --force <arg> --zip <arg> ```

  -- arg - game name / all(for all games in config.json)
